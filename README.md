# tinyrpc
## Tiny RPC
A high-performance RPC framework for personal learning, self encapsulated with logging modules and configuration modules.

## Operating environment
1. CentOS 8.2
2. g++(support C++11)
3. protobuf 3.19.4
4. tinyxml
